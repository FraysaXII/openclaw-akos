## Headline answer

AOS is a real but young, hype-saturated category (two shapes: kernel-level vs
coordination-layer). AKOS is a **KB-first coordination-layer AOS** that arrived
from the knowledge-base side — it passes the governance + audit test the field is
weakest on, and does **not** claim the kernel "OS" test it would fail. Honest
identity: *the operating system whose deliverables compound off a governed
knowledge base.* See `master-synthesis.md`.
