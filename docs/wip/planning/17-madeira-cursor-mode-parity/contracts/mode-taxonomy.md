# Madeira mode taxonomy (Ask / Plan / Run)

| User-visible label | `madeiraInteractionMode` (Madeira) | Cursor analogue | Escalation |
|--------------------|-------------------------------------|-----------------|------------|
| Lookup / Ask | `ask` | Ask | admin / exec → Orchestrator |
| Plan (draft) | `plan_draft` | Plan | Structured handoff → Orchestrator; mutations → swarm |
| Run | _(not a Madeira mode)_ | Agent | User switches to Orchestrator / swarm in dashboard |

**Run** is documented for UX parity; capability remains in **Executor + Verifier** with HITL gates.
