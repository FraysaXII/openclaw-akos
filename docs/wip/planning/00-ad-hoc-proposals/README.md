# Ad-hoc proposals (`00-ad-hoc-proposals/`)

**Not an initiative.** This folder holds experimental `.plan.md` files, model comparison notes, and one-off tooling proposals.

| Former path | Moved |
|:---|:---|
| `docs/wip/planning/99-proposals/` | **2026-06-13** → this folder |

**Why the rename:** Initiative **I99** (`99-supabase-platform-eg5-tranche/`) now uses the `99-` prefix for a governed program line. Keeping ad-hoc work under `99-proposals/` caused numbering confusion and would have blocked validators from scanning real I99 folders.

Numbered initiatives live under `docs/wip/planning/NN-<slug>/`. See [`../README.md`](../README.md).
