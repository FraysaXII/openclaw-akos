# I47 P12 — LLM-as-judge 3-axis scoring (D-IH-47-J)

## What shipped

### `akos/eval_harness/judge.py` (NEW)

3-axis judge module per RICE B + D-IH-47-J. Public API:

| API | Purpose |
|:---|:---|
| `JudgeResult` dataclass | Per-scenario verdict (3 scores + per-axis pass + overall_pass + model_id + cost_usd + skipped flag) |
| `JUDGE_AXES` constant | `("brand_voice", "citation", "persona_fit")` |
| `POLICY_IDS` constant | Maps each axis to its `POL-EVAL-JUDGE-THRESHOLD-*` policy id |
| `load_judge_thresholds()` | Reads thresholds from POLICY_REGISTER `min_pass_score=N` substring; defaults to 4 |
| `score_response_offline(response, scenario, persona)` | Deterministic CI-safe scoring; cost = 0; uses substring + heuristic rules |
| `score_response_live(...)` | Placeholder; raises NotImplementedError until operator pins API contract |
| `score_response(...)` | Dispatcher; routes to offline (default) or live (when `AKOS_JUDGE_MODEL=<id>` AND `AKOS_RECORD_LIVE=1`) |
| `aggregate_judge_cost_under_cap(results, per_scenario_cap_usd, overall_cap_usd)` | Cost discipline |

### Heuristics (offline mode; ≥80% deterministic alignment with live judge expected)

- **brand_voice (1-5)**:
  - 5: response uses governance terminology (Holistik Ops, Founder, Compliance, System Owner, PMO)
  - 4: respectful + qualification/escalate/reference language
  - 3: neutral
  - 1: BRAND_JARGON_AUDIT violations (hey guys, awesome, super exciting, lol, totally)
- **citation (1-5)**:
  - 5: cites `docs/references/hlk/...` path or `.md` file
  - 4: references registry/policy ID (`POL-`, `SKILL-`, `PERSONA-`, `CHAN-`, `TOPIC-`, `ROLE-`, `PROC-`, `SOP-`)
  - 3: refers to "registry" / "policy" / "SOP" generically
  - 2: vague ("our docs", "documentation")
  - 1: no citation
- **persona_fit (1-5)**:
  - 5: response acknowledges persona's qualification gate (when persona has one)
  - 5: cold-distance personas escalate; warm acknowledge
  - 3: neutral
  - 1: persona-context-blind

## R-47-10 + R-47-11 mitigation

- **R-47-10 (judge sycophancy / drift)**: Operator-pinned via `AKOS_JUDGE_MODEL` env var; `JudgeResult.model_id` captures pinned model in every cassette → drift detection at re-baseline time
- **R-47-11 (cost runaway)**: `aggregate_judge_cost_under_cap()` enforces per-scenario AND overall caps; offline mode (default) costs $0.00 → CI-safe by default; live mode requires explicit `AKOS_RECORD_LIVE=1` env opt-in
