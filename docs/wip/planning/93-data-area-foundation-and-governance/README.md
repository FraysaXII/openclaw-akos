# I93 — DATA Area Foundation & Cross-Area Data Governance

DATA is the only top-level area with no canonical doctrine. I93 builds it as a real,
DAMA-aligned, federated (data-mesh) governed area — and first mints a **People-governed
area-build-out meta-process** so the build-out (and every future area) is governable and
countable. DATA is the meta-process's first worked example; all other areas are then
harmonized to the same completeness bar.
