## Evidence quality bar

Per [`inline-ratify-craft/SKILL.md`](../../../.cursor/skills/inline-ratify-craft/SKILL.md) Principle 4 (cite evidence inline by file path and line range) — applies to every audit row produced at P1. Each substrate comparison row must cite:

- **External source URL + access date** (the vendor's official documentation or a credible third-party analysis).
- **Internal evidence path** when comparing to existing AKOS state (e.g., a `config/openclaw.json.example` field or a `config/agent-capabilities.json` role).
- **Founder principle alignment** when the substrate-choice intersects a founder principle (cited by `principle-2.X` per [`FOUNDER_METHODOLOGY_VERSIONING.md`](../../references/hlk/v3.0/Admin/O5-1/People/canonicals/FOUNDER_METHODOLOGY_VERSIONING.md)).
- **Historical decision when applicable** (e.g., the I10 D-IH-10-* decision that ratified Madeira's current tool surface; cited by row ID into DECISION_REGISTER.csv).

## Out-of-scope evidence

Explicitly NOT in I84 scope:

- **Live A/B comparison** between substrate options (e.g., running the same Madeira workflow on Cursor SDK + OpenClaw vs Cursor SDK alone vs OpenClaw alone). This is execution territory — defers to I76 P0 + I83 P0 owning initiatives per Q3 Option C shape-vs-execution split.
- **Cost-per-token measurement** at production scale. Defers to I52 multi-judge cost-discipline patterns; I84 surfaces cost-class enum + pricing-unit free-text, not measured cost.
- **Vendor-relationship outreach** (e.g., contacting Anysphere about enterprise terms). Defers to operator + Tech Lead direct engagement when D-IH-84-D MADEIRA productization shape concretes.
- **Code-level prototype** of any new substrate integration. Defers to each owning initiative's P0+ execution phases.
