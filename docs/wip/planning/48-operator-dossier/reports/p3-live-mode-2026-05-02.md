# I48 P3 — Live mode (CLI orchestrator <8min target; actual ~6.7s)

## What shipped

| Public symbol | Purpose |
|:---|:---|

### Section subclass live-mode wirings (8 sections)

Each Section's `gather()` checks mode and invokes the corresponding CLI when `mode in ("live", "tier-b")`; falls back to snapshot source otherwise. Per-CLI failure marks `placeholder=True` with stderr in error field — NEVER crashes the dossier.

| Section | Live behavior |
|:---:|:---|
| 2 | Snapshot CSV counts + `validate_hlk` PASS/FAIL verdict + duration |
| 3 | `eval --mode all --json` parsed; rows / modes_run / elapsed_ms / cost_total |
| 4 | Re-run `calibrate_scenarios.py` (writes fresh artifact); then read |
| 5 | `eval --mode adversarial --json` parsed; PII linter status |
| 6 | Snapshot read + `chaos --dry-run` gate-check (env-stubbed for safe execution; never live rotation) |
| 7 | `graphrag_drift_canary --csv-only` parsed for total drift count |
| 8 | Re-run `trigger_watcher` (writes fresh artifact); then read |
| 9 | Snapshot CSV (no live equivalent; weekly cadence) |
| 10 | Snapshot OPS-* parser (always re-parse; cheap) |

### `scripts/render_uat_dossier.py` extension

`--screenshots` opt-in flag now wires to `take_browser_screenshots()`; creates `screenshots/` subdir + README explaining operator workflow (the actual MCP-driven capture happens at UAT time, separately from the dossier render).

## R-48-1 graceful degradation verified

Per-CLI failure → `SectionData.placeholder=True` with stderr → markdown emits PLACEHOLDER text → dossier still renders + manifest still written + overall_status still computed. Test `test_run_cli_nonexistent_script_does_not_crash` proves the contract.

## Performance

P3 acceptance criterion: `--mode live` <8 minutes. Actual: **~6.7 seconds** for the full 10-CLI orchestration on operator laptop. The 5-minute estimate was conservative — the actual CLIs are fast (validate_hlk ~3s; eval --mode all ~2s; calibrate ~0.3s; rest <1s each). 5-minute budget reserved for future Tier B mode (P3+) and live LLM cassette recording.
