# Asset classification — Initiative 65

Per [`PRECEDENCE.md`](../../../references/hlk/compliance/PRECEDENCE.md).

## Canonical (proposed)

- **`governance.planning_workspace_view` Supabase view** — joins existing canonical mirrors (`compliance.initiative_registry_mirror`, `compliance.ops_register_mirror`, `compliance.decision_register_mirror`) plus computed columns (`reports_count_30d`, `last_report_at`, `freshness_state`). RLS: operator (level >= 4) read; service-role write. Migration ships in P1.

- **`governance.workspace_audit_log` (extension of `holistika_ops.audit_log`)** — does not need a new table; uses `action='planning.report.read'` etc. with `resource_kind='planning_initiative'` and `resource_id='<NN-slug>'`. Pure write extension; no new DDL.

## Mirrored / derived
