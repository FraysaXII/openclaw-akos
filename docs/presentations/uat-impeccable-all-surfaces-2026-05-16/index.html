The dual-register rule applies to vocabulary translation (internal CORPINT vocabulary vs external translated capability messaging); it does NOT apply to brand-name localization. The brand spelling is governed by CODE1 §"Plain prose form" + CODE2 §"Decision D-IH-66-A" which closes the spelling to Holistika (with K) universally; "Holistica" (with C) was rejected; "Holística" (with diacritic) is a logo-only stylization.
