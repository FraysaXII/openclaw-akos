## Purpose

Govern **agency and partner proposals** from any platform (not vendor-specific): **proposal → fit assessment → integration design → documentation → map to `process_list` / SOPs** for row **`holistika_gtm_dtp_002`**.

## Scope

External agencies, marketplaces, ad platforms, and vertical SaaS partners proposing co-sell or integration work.

## Procedure

1. **Intake:** Single intake channel (email or form); attach proposal and contact.
2. **Fit assessment:** Score against ICP, capacity, legal/regulatory fit (link Legal tasks in `process_list` when NDAs or DPAs required).
3. **Technical review:** HLK Tech Lab or designated architect for API/integration scope (see integration catalog initiative Phase 3).
4. **Decision:** Accept / negotiate / decline with written rationale (PMO optional witness).
5. **Register:** If accepted, add obligations to CRM and, when needed, new **External**-type process/tasks per governance—**no duplicate entities** in org baseline.

## Roles

- **Brand Manager / CMO:** Commercial owner of partner narrative.
- **Growth Manager:** Pipeline and co-marketing coordination.

## Execution runbook

### Per proposal (sequential gates)

- [ ] **Intake:** Log in single tracker (row ID); attach proposal PDF and primary contact.
- [ ] **Fit assessment:** Complete scorecard (ICP, capacity, legal); record **Pass / Negotiate / Decline**.
- [ ] **Technical review:** Schedule architect review within **5 business days** if Pass; else send decline template.
- [ ] **Decision:** CMO sign-off on commercial terms; Legal on NDAs/DPA when triggered.
- [ ] **Register:** If accepted, create CRM opportunity and link obligations; open `process_list` tranche only for **new** external obligations (no duplicate org entities).
