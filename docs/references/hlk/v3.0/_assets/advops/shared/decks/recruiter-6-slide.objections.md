# Recruiter Deck — Objection Companion

| Objection | Response Frame | Evidence |
|:---|:---|:---|
| Is this too early? | Be clear about scope, risk, and first 90 days. | Slide 04. |
| Will I have agency? | Ownership is artifact-based, not title theater. | Onboarding kit. |

| Is this serious? | Show canon, templates, and working systems. | Slide 02. |

## Operator Notes

- Hiring copy must not overpromise stability.
