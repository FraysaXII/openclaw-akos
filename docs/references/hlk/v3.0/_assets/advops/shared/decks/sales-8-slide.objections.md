# Sales Deck — Objection Companion

| Objection | Response Frame | Evidence |
|:---|:---|:---|
| How much will this cost? | Scope first; price follows clear deliverables. | Proposal template + service cells. |
| Will this become endless consulting? | Handoff is part of the method. | Engagement playbook transfer phase. |
| Do you understand my industry? | Discovery starts from the client's operating reality. | Discovery questionnaire + first workshop. |
| Is my data safe? | NDA/DPA before sensitive discovery. | Legal template suite. |

## Operator Notes

- Price ambiguity creates distrust. If pricing is not ready, say what determines it.
- Avoid using too much product terminology unless the buyer asked for technical implementation.
