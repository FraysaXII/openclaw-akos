# Program — `PRJ-HOL-FOUNDING-2026` (Legal chain)

**Owner role**: Legal Counsel  
**Program registry**: `process_list.csv` row for `PRJ-HOL-FOUNDING-2026` (founder incorporation program).  
**Scope**: All Legal-chain casework specifically scoped to the founder-incorporation program.  
**Forward layout convention**: [`docs/references/hlk/compliance/README.md`](../../../../../../../compliance/README.md) (Initiative 22 P1).

This folder is the **program-scoped landing point** for Legal casework on Holistika's founder-incorporation program. Its purpose is to keep program-specific case docs from polluting the role-folder root as more programs are added (`PRJ-HOL-KIRBE-*`, `PRJ-HOL-CLIENT-*`, etc.). New casework lands here; existing Legal-chain docs that are program-specific are listed below as **see-also** entries (they remain at the role-folder root for now to avoid link breakage).

> **Decision (D-IH-1, Initiative 22)** — Program subfolders are added going forward. Existing `FOUNDER_*` files **stay at the role-folder root** and gain a `> **Program**` admonition pointing here. Physical relocation is deferred to the same future initiative that consolidates program scope across all areas.
