# Methodology / Techniques

> **Research techniques** are the concrete, repeatable moves the pillars are built from — the
> toolbox a researcher reaches into during a cycle. This folder is **realized** (replaces the empty
> `Admin/O5-1/Research/Research Techniques/` husk per `D-IH-75-G`): it indexes the technique
> capabilities in
> [`CAPABILITY_REGISTRY.csv`](../../../Admin/O5-1/People/Compliance/canonicals/dimensions/CAPABILITY_REGISTRY.csv).
> Lifecycle: techniques span **ACQUIRE** (literature review, 6 Ws) and **PROCESS** (synthesis,
> benchmarking, packaging). Discipline: [`../README.md`](../README.md). Spine:
> [`../../canonicals/RESEARCH_LIFECYCLE_DOCTRINE.md`](../../canonicals/RESEARCH_LIFECYCLE_DOCTRINE.md).
