# Workflow: Browser Smoke Test

## Agent Sequence
1. **Verifier** -- browser-based validation

## Required Tools
- `browser_navigate`, `browser_snapshot`, `browser_screenshot`

## Steps
1. [ ] Navigate to dashboard URL
2. [ ] Verify page loads without errors
3. [ ] Check all expected agents are visible
4. [ ] Verify chat input is functional
5. [ ] Capture screenshot for evidence
6. [ ] Check browser console for errors

## Completion Criteria
- Dashboard loads successfully
- All expected elements present
- No console errors
- Screenshot captured
