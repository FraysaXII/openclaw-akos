# Workflow: Incident Review

## Agent Sequence
1. **Architect** -- root cause analysis
2. **Orchestrator** -- remediation plan

## Steps
1. [ ] Gather incident timeline from logs and traces
2. [ ] Identify root cause
3. [ ] Assess blast radius and affected components
4. [ ] Propose remediation steps
5. [ ] Document lessons learned
6. [ ] Update MEMORY.md with incident record

## Completion Criteria
- Root cause identified
- Remediation plan documented
- Lessons learned recorded
