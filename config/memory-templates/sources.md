# Memory Domain: Sources

Curated reference materials for evidence-backed retrieval.

## Entry Format

```
### [Source ID] -- [Title]
- **Type**: repo-file / external-url / memory-entry / operator-provided
- **URL or Path**: Location of the source
- **Freshness**: Date last verified
- **Credibility**: High / Medium / Low
- **Usage**: How this source has been used in past sessions
```
