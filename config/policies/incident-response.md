# Policy Pack: Incident Response

Governance profile for incident investigation and remediation.

## Rules
- All findings must be timestamped and logged.
- Root cause analysis must be completed before remediation.
- Remediation steps must be tested in dev-local before production.
- Incident record must be stored in memory/incidents/.
- Post-incident review must identify prevention measures.

## Applies To
- Architect agent (investigation)
- Orchestrator agent (coordination)
