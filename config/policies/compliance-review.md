# Policy Pack: Compliance Review

Governance profile for compliance-sensitive work (EU AI Act, SOC, audit).

## Rules
- All outputs must cite sources with fact IDs.
- Inferences must be clearly labeled as [INFERENCE].
- Decisions must be recorded in memory/decisions/.
- All tool calls must be logged to Langfuse.
- No external data ingestion without source credibility assessment.

## Applies To
- All agents
