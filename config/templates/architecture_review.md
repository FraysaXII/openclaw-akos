# Session Template: Architecture Review

## Goal
Analyze the codebase architecture and produce a structured assessment.

## Agent: Architect

## Suggested Prompts
- "Analyze the architecture of this codebase"
- "What are the main components and how do they interact?"
- "Identify architectural strengths and weaknesses"

## Expected Output
- Architecture diagram (text-based)
- Component inventory
- Strengths and gaps assessment
- Recommendations
